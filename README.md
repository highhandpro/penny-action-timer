# Penny Ante Action Timer

A simple dealer-facing penny ante action timer built as a static site for GitHub and Vercel.

## What’s included

- `index.html` - the app
- `DSEG7Classic-BoldItalic.ttf` - digital timer font
- `pennyante-og.png` - background artwork
- `vercel.json` - Vercel deployment settings

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. In Vercel, choose **New Project**.
3. Import the GitHub repo.
4. Deploy using the default static settings.

## Local use

Open `index.html` directly in a browser, or serve the folder with any static web server.
